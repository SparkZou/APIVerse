Integration Instructions

To integrate this widget into your website, add the following code:

<link rel="stylesheet" href="apiverse-widget/style.css">
<script src="apiverse-widget.js"></script>
<script>
  APIVerseWidget.init({
    apiKey: 'YOUR_API_KEY',
    knowledgeBaseId: 10,  // Your KB ID
    apiUrl: 'http://localhost:8000/api/widget'
  });
</script>
